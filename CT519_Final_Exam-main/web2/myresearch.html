Deep Learning

การเรียนรู้เชิงลึก (Deep Learning) การเรียนรู้เชิงลึกเป็นส่วนหนึ่งของการเรียนรู้ของเครื่องจักร (Machine Learning) มีพื้นฐานมาจากโครงข่ายประสาทเทียม (Artificial Neural Network : ANN) เป็นวิธีที่สร้างขึ้นเพื่อทำให้เครื่องจักรสามารถเรียนรู้และปรับตัวได้เองจากข้อมูลที่มีอยู่

โครงข่ายประสาทเทียมได้มีการพัฒนามาอย่างต่อเนื่องซึ่งวิธีที่นิยมใช้ในปัจจุบันคือ โครงข่ายประสาทคอนโวลูชัน (Convolution Neural Network: CNN), Long Short-Term Memory (LSTM), และ Recurrent Neural Networks (RNN).

บทที่ 5 สรุป อภิปรายและข้อเสนอแนะการวิจัย

5.1 สรุปผลการวิจัย

สรุปผลลัพธ์ที่สำคัญจากการวิจัย เช่น การใช้ Big Data ในการแก้ปัญหาการเดินทาง การพัฒนาระบบหรือแอปพลิเคชันที่ให้ข้อมูลการเดินทางแบบเรียลไทม์ และการพยากรณ์การจราจร

5.2 อภิปรายผลการวิจัย

อภิปรายผลลัพธ์ที่ได้จากการวิจัย เช่น ข้อดีและข้อจำกัดของการใช้ Big Data ในการเดินทาง ประสิทธิภาพและความแม่นยำของระบบหรือแอปพลิเคชัน และการปรับปรุงอัลกอริทึมในการคำนวณเส้นทางและการพยากรณ์การจราจร

5.3 ข้อเสนอแนะการวิจัย

เสนอแนวทางในการปรับปรุงและพัฒนาการใช้ Big Data ในการเดินทางในอนาคต เช่น การใช้เทคโนโลยีใหม่ ๆ การปรับปรุงกระบวนการรวบรวมและวิเคราะห์ข้อมูล และการพัฒนาระบบหรือแอปพลิเคชันที่มีประสิทธิภาพมากขึ้น
